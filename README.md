 🏏 IPL Cricket Analytics Project

An end-to-end Data Analytics and Machine Learning project based on Indian Premier League (IPL) cricket data. This project analyzes team performance, player statistics, match trends, toss impact, venue insights, and predicts match outcomes using Python and Machine Learning techniques.

---
 📌 Project Overview

The main objective of this project is to perform data analysis on IPL datasets and extract meaningful insights from historical cricket matches.

This project includes:

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Team Performance Analysis
- Player Statistics Analysis
- Toss Impact Analysis
- Venue-Based Insights
- Batting & Bowling Analytics
- Match Winner Prediction
- Interactive Data Visualization

---

 🚀 Features

✔ Top Run Scorers Analysis  
✔ Top Wicket Takers Analysis  
✔ Team Win Percentage  
✔ Toss Impact on Match Results  
✔ Venue Performance Analysis  
✔ Strike Rate & Economy Rate Analysis  
✔ Match Winner Prediction Model  
✔ Interactive Charts & Graphs

 📂 Dataset

The project uses IPL datasets containing:

- Match-level data
- Ball-by-ball delivery data

Dataset files:
- `matches.csv`
- `deliveries.csv`

Dataset Source:
- Kaggle IPL Dataset

 ▶️ Installation & Setup

 Clone the Repository

```bash
git clone https://github.com/your-username/ipl-cricket-analytics.git
```

 Navigate to Project Folder

```bash
cd ipl-cricket-analytics
```

 Install Required Libraries

```bash
pip install -r requirements.txt
```
 Run Jupyter Notebook

```bash
jupyter notebook
```

### Run Streamlit App

```bash
streamlit run app.py
```
